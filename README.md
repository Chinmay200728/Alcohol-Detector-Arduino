# 🍺 Alcohol Detector Using Arduino Uno

An Arduino-based alcohol detection system that uses the MQ-3 sensor to detect alcohol vapors in the surrounding environment and provide real-time monitoring.

## 📖 Project Overview

The Alcohol Detector project is designed to detect the presence of alcohol in the air using an MQ-3 Alcohol Sensor connected to an Arduino Uno. The sensor continuously monitors alcohol concentration and sends data to the Arduino for processing. This project demonstrates the practical application of embedded systems and sensor technology in safety monitoring.

## 🎯 Objectives

- Detect alcohol vapors in real time
- Monitor sensor readings using Arduino Uno
- Provide a simple and low-cost safety solution
- Learn sensor interfacing and embedded system design

## 🛠️ Components Used
